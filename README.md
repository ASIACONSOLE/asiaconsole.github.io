# AsiaConsole (formerly TechCom)

AsiaConsole, teknoloji, oyun ve mobil uygulama dünyasına hitap eden modern ve dinamik bir topluluk platformudur.

## 🚀 Özellikler

-   **Gelişmiş CMS**: Admin paneli üzerinden sayfa ve içerik yönetimi.
-   **Canlı Yapay Zeka**: Google Gemini entegrasyonu ile akıllı asistan desteği.
-   **Kullanıcı Paneli**: Üyelik sistemleri ve yetkilendirme katmanları.
-   **Forum & Sosyal Etkileşim**: Moderasyon destekli topluluk tartışmaları.
-   **Dinamik Tema Sistemi**: 10'dan fazla hazır tema ve özelleştirilebilir tasarım seçenekleri.

## 🛠️ Kurulum

1.  Repoyu bilgisayarınıza indirin.
2.  `index.html` dosyasını tarayıcınızda açın.
3.  Admin paneline erişmek için `/admin/index.html` yolunu kullanın (Varsayılan: admin / admin123).

## 🤖 Yapay Zeka Entegrasyonu

Site içerisinde gerçek zamanlı yapay zeka kullanmak için Admin Paneli -> Site Ayarları kısmından kendi **Gemini API Key**'inizi ekleyebilirsiniz.
